# 📚 Jetpack Compose UI Guides Overview

Welcome to the Jetpack Compose UI component study notes. This index serves as your master reference for exploring the various components available in Jetpack Compose, structured by category. 

## 🗂️ Categories

1. 🔤 [[01-Text-Input]] — Input fields, search bars, and dropdowns.
2. ✅ [[02-Selection-Controls]] — Checkboxes, radio buttons, switches, and sliders.
3. 🔘 [[03-Buttons]] — Action buttons, FABs, icon buttons, and button groups.
4. 🍟 [[04-Chips]] — Compact elements that represent an input, attribute, or action.
5. 📜 [[05-Lists]] — Lazy columns, rows, and grids.
6. 🗂️ [[06-Surfaces]] — Cards, surfaces, and elevation.
7. 🧭 [[07-Navigation]] — Bottom navigation, navigation rails, and drawers.
8. 🪟 [[08-Dialogs]] — Alert dialogs, bottom sheets, and custom modals.
9. 📊 [[09-Progress]] — Linear and circular progress indicators.
10. 🖼️ [[10-Media]] — Images, icons, and async image loading.
11. 📐 [[11-Layouts]] — Rows, Columns, Boxes, and complex layouts.
12. 🔔 [[12-Notifications]] — Snackbars, toasts, and in-app messaging.
13. 🎨 [[13-Theming]] — Colors, typography, shapes, and dark mode.
14. ✨ [[14-Animation]] — Animated visibility, transitions, and state changes.
15. 🎛️ [[15-AppBars]] — Top and bottom app bars.
16. 🎭 [[16-Menus]] — Context menus and popup menus.

> [!TIP] 
> Start with the basic components and gradually move on to the complex interactive ones. Each folder contains detailed `.md` files covering component signatures, props, and real-world examples.
