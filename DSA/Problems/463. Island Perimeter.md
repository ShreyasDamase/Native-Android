## :LiLayoutPanelLeft:  Pattern Name: DP

## Qeustion:

You are given `row x col` `grid` representing a map where `grid[i][j] = 1` represents land and `grid[i][j] = 0` represents water.

Grid cells are connected **horizontally/vertically** (not diagonally). The `grid` is completely surrounded by water, and there is exactly one island (i.e., one or more connected land cells).

The island doesn't have "lakes", meaning the water inside isn't connected to the water around the island. One cell is a square with side length 1. The grid is rectangular, width and height don't exceed 100. Determine the perimeter of the island.

## Key Idea

- iterate over the 2s grid when we have value of cell 1 then do dfs search of cell val 1
- return count from dfs and add it to count 
- base condition
- 1) hadle boundary condition so if i < 0 0r i >= row size or j < 0 or j>= column size then return 1 as we have valid perimeter means side 
- 2) if we found that cell is 0 that is g[i][j] also return 1 as it is also a valid perimeter 
- 3) if we found cell as -1 mean we already visited and it is adjoined cell to previous so it will not get count as perimeter 
- mark current cell as -1 as we visited 
- call next dfs     with return this value as we need to add thir result `return dfs(g,i+1,j)+dfs(g,i-1,j)+dfs(g,i,j+1)+dfs(g,i,j-1)`

## Template (Kotlin)
```kotlin
class Solution {

    fun islandPerimeter(grid: Array<IntArray>): Int {
        if (grid.isEmpty()) return 0

        var count = 0

        for (i in grid.indices) {
            for (j in grid[i].indices) {

                if (grid[i][j] == 1) {
                    val temp = dfs(grid, i, j)
                    count += temp
                }

            }
        }

        return count
    }

    fun dfs(g: Array<IntArray>, i: Int, j: Int): Int {

        // Boundary → contributes to perimeter
        if (i < 0 || j < 0 || i >= g.size || j >= g[0].size) {
            return 1
        }

        // Already visited
        if (g[i][j] == -1) {
            return 0
        }

        // Water → contributes to perimeter
        if (g[i][j] == 0) {
            return 1
        }

        // Mark visited
        g[i][j] = -1

        // Sum all 4 directions
        return dfs(g, i + 1, j) +
               dfs(g, i - 1, j) +
               dfs(g, i, j + 1) +
               dfs(g, i, j - 1)
    }
}
